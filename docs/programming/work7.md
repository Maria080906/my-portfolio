## Что такое перегрузка?

**Перегрузка (Overloading)** — это возможность использовать одно и то же имя функции или метода для разных типов или количества аргументов.

В Python перегрузка реализуется с помощью:
- Параметров по умолчанию
- `*args` (произвольное количество аргументов)
- `isinstance()` (проверка типа)

## Пример перегрузки из репозитория

В папке `contrib/overload/` находится пример, демонстрирующий перегрузку методов в Python:

```python
class OverloadDemo:
    def show(self, *args):
        if len(args) == 1:
            print(f"Один аргумент: {args[0]}")
        elif len(args) == 2:
            print(f"Два аргумента: {args[0]} и {args[1]}")
        else:
            print(f"Аргументов: {len(args)} -> {args}")

    def greet(self, name, greeting="Привет"):
        print(f"{greeting}, {name}!")
